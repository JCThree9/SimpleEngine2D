Instructions for launching project:
Follow the following steps
1: Install the .NET 9 SDK — download from https://dotnet.microsoft.com/download
2: Install Git — https://git-scm.com/downloads
3: Open terminal and navigate to file you want to have game engine in
4: git clone https://github.com/JCThree9/SimpleEngine2D
5: cd SimpleEngine2D
6: dotnet restore
7: dotnet new tool-manifest
8: dotnet tool install dotnet-mgcb
9: dotnet tool install dotnet-mgcb-editor
10: dotnet build
In order to run game engine: dotnet run --project ProjectHub
